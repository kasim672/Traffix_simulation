# requirements.txt

## Purpose
Lists Python package dependencies required to run the project.

## Current Dependency
- `pygame>=2.0.0`

## Why It Is Needed
Pygame provides:
- window and display management,
- input event handling,
- timing utilities,
- 2D drawing primitives,
- font rendering.

## Installation
From the project root:

```bash
pip install -r requirements.txt
```
